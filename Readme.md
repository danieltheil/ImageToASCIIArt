# ImageToASCIIArt

# Requires:
* python
* pip
* PIL

# How to Run:
* run install.bat
* py imgToAscii.py <path_to_img> <int: resolution (best results 500 - 1000)>
* ascii image will be deposited in the out folder# ImageToASCIIArt
